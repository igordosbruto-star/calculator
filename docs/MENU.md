# Menu e Navegação

Estrutura de menus disponível na linha de comando.

## Menu Principal

- **Materiais**
  - Listar
  - Criar
- **Projetos**
  - Abrir
  - Listar
- **Relatórios**
  - Custos
  - Tempos

A navegação é realizada por meio de subcomandos, por exemplo `./app listar` ou `./app comparar --ids=1,2`.
